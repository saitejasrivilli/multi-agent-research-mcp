# Multi-Agent Research Assistant

AI-powered research using autonomous agents (Researcher → Critic → Synthesizer).

## Tech Stack
- Llama 3.3 70B (Groq)
- Tavily Search
- Multi-Agent Architecture
- Streamlit

## Live Demo
https://multi-agent-research-mcp.streamlit.app

## Setup
1. Clone repo
2. Create .streamlit/secrets.toml with API keys
3. pip install -r requirements.txt
4. streamlit run app.py
